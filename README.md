# OperatingSystem
操作系统实验与课设

## 实验

- **进程调度**
- **银行家算法**
- **动态分区分配方式的模拟**
- **仿真各种磁盘调度算法**

## 课设

使用Visual Studio2019+QT5.14.2

- **提交一批作业（>=10），按先来先服选择一部分作业（最多5个）进入内存**
- **为每个作业创建一个进程，并分配内存（用户内存：0—1024K，采用可变连续分配方式）**
- **进程调度功能（时间片轮转）**
- **随机阻塞进程，并在一段时间后唤醒进程（选做）**
- **显示相关信息：后备作业队列、内存分配情况、进程信息、完成作业情况**
- **这些功能要有机地连接起来**
